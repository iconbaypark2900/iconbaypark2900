My rule is simple: **a claim should resolve to an artifact** — a dataset, structure file, test result, decision record, circuit, benchmark, or preserved failure.

## How I work

```text
question
→ reproducible input
→ system or experiment
→ baseline
→ measured output
→ failure record
→ promotion gate
```

I prefer:

- reproducible experiments over polished claims;
- explicit classical and quantum backend provenance;
- paper trading before capital deployment;
- independent coding agents with human approval gates;
- local-first AI systems that can run across my DGX Spark and EVO-X2 workstations;
- failures preserved as evidence instead of quietly discarded.

## Current research threads
